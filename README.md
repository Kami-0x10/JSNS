node js、npm、psqlが動くサーバ<br>
htmlクライアントサーバ(できればあったほうがいい)<br>
ブラウザ<br>
以下必須パッケージ(npm installしちゃってください)<br>
<br>
├── bcryptjs<br>
├── body-parser<br>
├── cors<br>
├── express<br>
├── fs<br>
├── init<br>
├── jsonwebtoken<br>
├── multer<br>
├── path<br>
└── pg<br>
